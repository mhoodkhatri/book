# Development Workstation

This guide covers the hardware requirements for your development workstation.

## Minimum Requirements

| Component | Minimum | Recommended |
|-----------|---------|-------------|
| CPU | 8 cores, 3.0 GHz | 16+ cores, 3.5+ GHz |
| RAM | 16 GB | 32+ GB |
| GPU | NVIDIA GTX 1660 | NVIDIA RTX 3080+ |
| Storage | 256 GB SSD | 1+ TB NVMe SSD |
| OS | Ubuntu 22.04 LTS | Ubuntu 22.04 LTS |

## GPU Requirements

A CUDA-capable NVIDIA GPU is essential for:

- Isaac Sim and Isaac ROS
- Deep learning inference
- Real-time simulation rendering

### Verified GPUs

- NVIDIA RTX 3080, 3090, 4080, 4090
- NVIDIA RTX A4000, A5000, A6000
- NVIDIA Quadro series

## Software Prerequisites

Before starting the course, install:

1. **Ubuntu 22.04 LTS** - The primary development OS
2. **NVIDIA Drivers** - Latest stable version
3. **CUDA Toolkit 12.x** - For GPU acceleration
4. **Docker** - For containerized development
5. **ROS 2 Humble** - Robot Operating System
